ManaParcel System

Project Overview

The ManaParcel System is a parcel tracking and management platform developed for a Malaysia-based startup established in Ipoh, Perak, in April 2023. The system caters to the needs of individuals and businesses seeking reliable and effective parcel tracking services. It aims to enhance the shipping and delivery experience in Malaysia by offering real-time updates and an intuitive user interface.

Key Features

  • Real-Time Parcel Tracking: Users can enter their parcel ID to view the current location, status, and expected delivery time.
  • Parcel Management Dashboard: Admin users can add, update, delete, and search for parcel records.
  • Notifications: Provides updates on parcel status changes (e.g., "Out for Delivery," "Delivered").
  • Search and Filter: Advanced search functionality for users to locate parcels based on tracking ID, status, or date.

Role and Contributions

  • Designed the database schema to efficiently store parcel tracking details such as parcel ID, current location, status, and timestamps.
  • Designed an interactive dashboard for users to view parcel details and track progress.
  • Built responsive web pages for tracking, managing, and searching parcels using HTML, CSS, and JavaScript.
  • Debugged issues related to API integration and database queries.

Challenges Faced and Solutions

  • Challenge: Designing an Intuitive and User-Friendly Interface.
    Solution:Conducted usability testing with potential users and iterated the design based on feedback.
  • Challenge: Ensuring Real-Time Updates from Courier APIs.
    Solution: Implemented efficient API polling mechanisms to retrieve updates and optimized database storage for fast retrieval.

Tools and Technologies

  • Front-End: HTML, CSS, JavaScript
  • Database: MySQL for data storage and retrieval of parcel tracking details
  • PHP
  • XAMPP
