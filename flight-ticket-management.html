Flight Ticket Management System

Project Overview

The Flight Ticket Booking System is a user-friendly platform designed to simplify the process of booking airline tickets. It enables customers to search for and book flights by entering essential details, offering features such as one-way or round-trip options, business/economy class selection, and transparent pricing.

Key Features

  • Flight Information Management: Captures details such as flight number, origin-destination, boarding date/time, price, and type (business/economy).
  • Trip Type Selection: Allows booking of one-way or round-trip tickets.
  • Price Calculation and Receipt Generation: Automatically calculates prices and displays detailed receipts.
  • Customer-Friendly Interface: Simplifies entering travel details and viewing available options.

Role and Contributions

  • Designed and implemented the core logic for flight search and ticket booking using C++.
  • Built the receipt generation feature, including tax calculations and trip-specific pricing.
  • Developed a modular, reusable backend architecture for better maintainability.
  • Conducted testing to ensure the accuracy of pricing calculations and smooth handling of round-trip bookings.

Challenges Faced and Solutions

  • Challenges: Calculating ticket prices dynamically based on trip type (one-way/round-trip) and class (business/economy) while accounting for tax rates was computationally intensive.
  • Solution: Created a flexible pricing function using lambda functions to dynamically adjust based on trip type and class.
  • Challenges: Displaying detailed receipts dynamically, including trip type, ticket class, tax, and total, required a well-structured format.
  • Solution: Used formatted output in C++ with std::stringstream for clear and printable receipts.
